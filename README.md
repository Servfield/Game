# 玄穹·问道（Xuanqiong · Wendao）

一款**高级感暗金风格**的修仙网页小游戏：
- 轻放置（吐纳自动产出灵气与道点）
- Roguelite 秘境行（战斗/奇遇/商队/机缘/休整，终层魇主）
- 闭关突破（多变量概率模型，成败皆命）
- 功法玄图（小型技能树，影响吐纳、秘境与突破）
- 炼丹/炼器（配方系统、品质、耐久）
- 自动存档（localStorage）+ 导出/导入
- **纯静态前端**：无需构建、无需后端，直接上 GitHub Pages

> 设计目标：不 low、耐玩、节奏有张力，UI 高端大气。

---

## 1) 本地运行

直接双击 `index.html` 或用任意静态服务器：

```bash
# Python 3
python -m http.server 5173
```

然后浏览器打开：`http://localhost:5173/`

---

## 2) GitHub Pages 一键部署（推荐）

1. 新建一个 GitHub 仓库（Public 最方便）。
2. 把本项目文件上传到仓库根目录。
3. 在仓库 Settings → Pages：
   - Source 选择 `Deploy from a branch`
   - Branch 选择 `main`（或 `master`）
   - Folder 选择 `/ (root)`
4. 保存后等几十秒，Pages 地址就会出现。

> 如果你更喜欢 Actions 部署，也可以自行添加 workflow；本项目已内置 Service Worker 以支持离线缓存。

---

## 3) 玩法速成

- **起步**：先点「打坐吐纳」几次攒灵气，然后「升阶灵阵」。
- **玄图**：攒道点后优先参悟「太初吐纳」「星纹聚灵阵」。
- **秘境**：资源紧缺就进入「星渊行」，优先拿灵材与残页。
- **突破**：在灵气与灵材充足时点「闭关突破」。悟性/气运/神识/灵阵阶越高越稳。
